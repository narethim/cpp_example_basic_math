# cpp_example_basic_math - Cpp Example code and Unit Test using CppUnit

## Prerequisite

* Java SDK Development
* Apache ant
* CppUnit library

## Directory structure

```sh

cpp_example_basic_math
  |
  + src ( cpp files and Makefile )
  |
  + test_cppunit
    |
    + src ( Unit Test files and Makefile )

```

## How to run it

```sh

cd cpp_example_basic_math

ant

ant run

ant clean

ant run-test

```
